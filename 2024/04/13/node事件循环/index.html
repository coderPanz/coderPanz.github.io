д»Ӣз»Қ

вҖӢ Node.js дәӢд»¶еҫӘзҺҜпјҲEvent LoopпјүжҳҜ Node.js е®һзҺ°ејӮжӯҘ I/O ж“ҚдҪңзҡ„ж ёеҝғжңәеҲ¶пјҢе®ғиҙҹиҙЈеӨ„зҗҶдәӢд»¶е’Ңжү§иЎҢеӣһи°ғеҮҪж•°пјҢд»Ҙе®һзҺ°йқһйҳ»еЎһ I/O ж“ҚдҪңе’ҢејӮжӯҘзј–зЁӢгҖӮNode.js дәӢд»¶еҫӘзҺҜзҡ„е®һзҺ°еҹәдәҺ libuv еә“пјҢе®ғжҳҜдёҖдёӘи·Ёе№іеҸ°зҡ„ејӮжӯҘ I/O еә“пјҢз”ЁдәҺеӨ„зҗҶдәӢд»¶еҫӘзҺҜгҖҒж–Үд»¶зі»з»ҹж“ҚдҪңгҖҒзҪ‘з»ңиҜ·жұӮзӯүгҖӮ

вҖӢ NodeJS дёӯеҢ…еҗ«зҡ„ејӮжӯҘжңәеҲ¶еңЁе’ҢжөҸи§ҲеҷЁдёӯзҡ„зұ»дјјгҖӮйҷӨжӯӨд№ӢеӨ–пјҢNode.jsдёӯиҝҳжңүдёҖдәӣе…¶д»–зҡ„ејӮжӯҘеҪўејҸйңҖиҰҒдҫқйқ nodejsзү№жңүзҡ„дәӢд»¶еҫӘзҺҜжңәеҲ¶дәҶгҖӮ

  • ж–Үд»¶иҜ»еҸ–зҡ„I/Oж“ҚдҪңпјҡйҖҡиҝҮfsжЁЎеқ—еҸҜе®һзҺ°ејӮжӯҘIOж“ҚдҪңгҖӮ
  • process.nextTick()пјҡеңЁжҹҗдәӣеҗҢжӯҘд»»еҠЎе®ҢжҲҗеҗҺз«Ӣй©¬жү§иЎҢгҖӮ
  • server.closeгҖҒhttpServerзӯүдёҖдәӣжңҚеҠЎзҡ„е…ій—ӯпјҡе…ій—ӯеӣһи°ғгҖӮ
  • setImmediate()пјҡдёҺ setTimeout еңЁжҹҗдәӣеҗҢжӯҘд»»еҠЎе®ҢжҲҗеҗҺз«Ӣй©¬жү§иЎҢгҖӮ

nodeдәӢд»¶еҫӘзҺҜжөҒзЁӢ

nodeдәӢд»¶еҫӘзҺҜеңЁlibuvзҡ„и°ғжҺ§дёӢеҲҶдёә6дёӘиҝҮзЁӢпјҢе№¶еҫӘзҺҜжү§иЎҢиҝҷдёӘиҝҮзЁӢгҖӮ

  • timers пјҡsetTimeoutгҖҒsetIntervalзҡ„еӣһи°ғпјҢе№¶дё”иҜҘжҳҜеңЁpoll йҳ¶ж®өеӣһи°ғжқҘжү§иЎҢзҡ„пјӣ

  • I/O ж“ҚдҪңзҡ„еӣһи°ғпјҡдё»иҰҒжү§иЎҢзі»з»ҹзә§еҲ«зҡ„еӣһи°ғеҮҪж•°пјҢж–Үд»¶иҜ»еҸ–еӣһи°ғгҖҒжңҚеҠЎз«ҜеҸЈзҡ„зӣ‘еҗ¬еӣһи°ғпјӣ

  • idle, prepare йҳ¶ж®өпјҡд»…Node.jsеҶ…йғЁжү§иЎҢиҝҮзЁӢпјӣ

  • poll йҳ¶ж®өпјҡиҪ®иҜўзӯүеҫ…ж–°зҡ„й“ҫжҺҘе’ҢиҜ·жұӮзӯүдәӢд»¶пјҢжү§иЎҢ I/O еӣһи°ғзӯүпјӣ

  • check йҳ¶ж®өпјҡжү§иЎҢ setImmediate() еӣһи°ғпјӣ

  • close еӣһи°ғ йҳ¶ж®өпјҡжү§иЎҢе…ій—ӯиҜ·жұӮзҡ„еӣһи°ғеҮҪж•°пјҢжҜ”еҰӮsocket.on(вҖҳcloseвҖҷ, вҖҰ)

вҖӢ дёҠиҝ°жҜҸдёӘйҳ¶ж®өйғҪдјҡеҺ»жү§иЎҢе®ҢеҪ“еүҚйҳ¶ж®өеҜ№еә”зҡ„д»»еҠЎйҳҹеҲ—пјҢз„¶еҗҺз»§з»ӯжү§иЎҢеҪ“еүҚйҳ¶ж®өзҡ„еҫ®д»»еҠЎйҳҹеҲ—пјҢеҸӘжңүеҪ“еүҚйҳ¶ж®өжүҖжңүеҫ®д»»еҠЎйғҪжү§иЎҢе®ҢдәҶпјҢжүҚдјҡиҝӣе…ҘдёӢдёӘйҳ¶ж®өгҖӮиҝҷдёҺжөҸи§ҲеҷЁдәӢд»¶еҫӘзҺҜдёҚеҗҢгҖӮе…¶дёӯpollйҳ¶ж®өдјҡеӣһеҲ°timeersйҳ¶ж®өжү§иЎҢеӣһи°ғе’ҢI/Oеӣһи°ғгҖӮ

process.nextTick()

вҖӢ дёҠйқўжҸҗеҲ°дәҶprocess.nextTick()пјҢе®ғжҳҜnodeдёӯж–°еј•е…Ҙзҡ„дёҖдёӘд»»еҠЎйҳҹеҲ—пјҢе®ғдјҡеңЁдёҠиҝ°еҗ„дёӘйҳ¶ж®өз»“жқҹж—¶пјҢеңЁиҝӣе…ҘдёӢдёҖдёӘйҳ¶ж®өд№ӢеүҚз«ӢеҚіжү§иЎҢгҖӮеңЁ Node.js дёӯпјҢprocess.nextTick() ж–№жі•е…Ғи®ёдҪ еңЁеҪ“еүҚжү§иЎҢж Ҳе®ҢжҲҗеҗҺз«ӢеҚіжү§иЎҢдёҖдёӘеӣһи°ғеҮҪж•°гҖӮиҝҷдёӘеӣһи°ғеҮҪж•°дјҡеңЁеҪ“еүҚдәӢд»¶еҫӘзҺҜзҡ„жң«е°ҫжү§иЎҢпјҢеңЁд»»дҪ• I/O ж“ҚдҪңд№ӢеүҚпјҢеҚідҪҝиҝҷдёӘ I/O ж“ҚдҪңжҳҜејӮжӯҘзҡ„гҖӮиҝҷдҪҝеҫ— process.nextTick() зҡ„жү§иЎҢдјҳе…Ҳзә§й«ҳдәҺ setTimeout() е’Ң setImmediate()гҖӮ

е®Ҹд»»еҠЎе’Ңеҫ®д»»еҠЎ

Node.jsдәӢд»¶еҫӘзҺҜзҡ„ејӮжӯҘйҳҹеҲ—д№ҹеҲҶдёәдёӨз§Қпјҡе®Ҹд»»еҠЎйҳҹеҲ—е’Ңеҫ®д»»еҠЎйҳҹеҲ—гҖӮ

  • еёёи§Ғзҡ„е®Ҹд»»еҠЎпјҡI/O ж“ҚдҪңгҖҒsetTimeoutгҖҒsetIntervalгҖҒ setImmediateзӯүгҖӮ
  • еёёи§Ғзҡ„еҫ®д»»еҠЎпјҡprocess.nextTickгҖҒpromise().then()зӯүгҖӮ

setImmediate е’Ң setTimeout

setImmediate е’Ң setTimeoutпјҢиҝҷдёӨиҖ…еҫҲзӣёдјјпјҢдё»иҰҒеҢәеҲ«еңЁдәҺи°ғз”Ёж—¶жңәзҡ„дёҚеҗҢпјҡ

  • setImmediateпјҡеңЁpollйҳ¶ж®өе®ҢжҲҗж—¶жү§иЎҢпјҢеҚіcheckйҳ¶ж®өпјӣ

  • setTimeoutпјҡеңЁpollйҳ¶ж®өдёәз©әй—Іж—¶пјҢдё”и®ҫе®ҡж—¶й—ҙеҲ°иҫҫеҗҺжү§иЎҢпјҢдҪҶе®ғеңЁtimerйҳ¶ж®өжү§иЎҢпјӣ
